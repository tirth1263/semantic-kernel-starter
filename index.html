Semantic Kernel Starter

A compact, production-minded starter for building a streaming ChatCompletionAgent with custom plugins and Nebius Token Factory.

GitHub Quickstart

What it includes

Everything needed to test an agentic Semantic Kernel flow.

Streaming Agent

Uses ChatCompletionAgent with streamed responses for a responsive CLI experience.

Custom Plugin

Includes a TimePlugin that demonstrates the @kernel_function decorator.

Tool Calling

Enables automatic function choice so the model decides when to call tools.

Nebius Ready

Connects through Semantic Kernel's OpenAI-compatible service path.

Run it

Clone, configure, and start chatting.

Installation 
git clone https://github.com/tirth1263/semantic-kernel-starter.git
cd semantic-kernel-starter

pip install -r requirements.txt
Environment 
cp .env.example .env
# set NEBIUS_API_KEY

python main.py

How it works

One kernel, one agent, one useful plugin.

The app builds a Semantic Kernel, registers an OpenAI-compatible Nebius chat service, adds the TimePlugin, and gives the agent automatic function-calling settings. When the user asks for the current time, the model can call time.now instead of guessing.
